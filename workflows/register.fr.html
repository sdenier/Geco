Inscriptions

Cette page résume les différentes possibilités pour inscrire des concurrents dans une course. Aucune option n’est exclusive : il est toujours possible de mixer ces méthodes suivant les besoins en cours.

Pré-inscription des concurrents

Il existe deux méthodes pour pré-inscrire facilement des concurrents avant la compétition.

  • Vous pouvez importer un fichier CSV avec le format des inscriptions OE. C’est la méthode la plus rapide pour importer de nombreuses inscriptions. Le fichier peut être généré et édité avec n’importe quel tableur.
  • Vous pouvez importer des coureurs depuis une archive pour OE, créée par une fédération ou les clubs.

Attention ! Avec le format OE CSV original, il n’est pas possible d’indiquer directement les circuits des coureurs. À la place, il faut utiliser la catégorie pour inscrire le coureur sur son circuit. Avant l’import, il faut donc créer les circuits et les catégories dans la configuration, puis associer les circuits aux catégories. À défaut les coureurs seront sur le circuit [Auto]. Il existe cependant une version spéciale de ce format permettant l’attribution directe des circuits (voir ci-dessous).

L’import CSV ou par l’archive crée automatiquement les clubs et catégories nécessaires pour l’import.

Dans le format OE CSV, seuls les champs Nom, Prénom, le club et la catégorie (court/long) sont obligatoires. Le numéro de départ, la puce, l’identifiant archive, le champ NC et l’heure de départ sont tous optionnels. Les autres champs sont ignorés par Geco. Les contraintes sont les mêmes pour le format de l’archive, avec quelques champs en moins.

Le champ Heure de départ permet de donner un horaire pré-attribué, sans utiliser de boîtier départ. L’heure donnée doit être relative à l’heure zéro de la course.

Pré-inscription des concurrents avec circuits pré-attribués

Geco 2 peut aussi importer une version spéciale du format OE CSV contenant déjà les circuits attribués aux coureurs. Ce format permet par exemple d’exporter les séries de qualification comme liste de départ pour la prochaine course. Cette astuce utilise deux colonnes non exploitées par Geco dans le format original :

  • colonne ‘Temps/Time’ taggé avec le mot-clé ‘Geco-course’
  • colonne ‘Évaluation/Eval’ : nom du circuit attribué au coureur

Lors de l’import d’une liste de départ, Geco détecte pour chaque entrée le tag ‘Geco-course’ : s’il est présent, Geco traite la colonne suivante comme le nom du circuit pour le coureur. Sinon, Geco revient au mode classique en inscrivant le coureur sur le circuit de sa catégorie ou le circuit [Auto] par défaut.

Inscription sur site

Pour les inscriptions le jour de la course, plusieurs options sont possibles :

  • vous pouvez insérer une nouvelle entrée en cliquant le bouton + (raccourci mod+I) et éditer l’entrée manuellement ;
  • vous pouvez utiliser l’archive, comme ci-dessus ;
  • vous pouvez utiliser le mode inscription, qui permet d’enregistrer directement le numéro de puce et fait automatiquement une recherche dans l’archive.

Inscription à la volée

Avec le process auto de Geco, il est possible de minimiser les pré-inscriptions, voire de s’en passer pour les organisation légères. Geco créera les entrées au fur et à mesure lors de la lecture des puces. L’inscription à la volée peut se faire indifféremment en mode compétition ou entraînement.
