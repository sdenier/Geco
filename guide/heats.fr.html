Onglet Séries

L’onglet séries permet de définir les séries de qualification (par élimination directe) pour la prochaine étape. Geco calcule les coureurs qualifiés à partir des résultats et des spécifications de séries. Les séries peuvent être exportées directement comme liste de départ pour l’étape suivante.

Un groupe de séries décrit les conditions de qualification des coureurs. Pressez le bouton Créer pour créer un nouveau groupe, ou bien éditez un groupe existant en double-cliquant dessus dans la liste. L’Éditeur de séries s’affiche et vous permet de configurer les paramètres pour ce groupe :

  • donnez un nom au groupe
  • donnez les noms des séries des coureurs qualifiés (séparés par une virgule). Les séries correspondent aux circuits de l’étape suivante
  • donnez le rang de qualification (classement à atteindre pour être qualifié)
  • indiquez si les coureurs doivent être classés par circuit, catégorie, ou la combinaison catégorie/circuits

Sauvez les paramètres du groupe. Finalement, sélectionnez dans la liste quels circuits (ou catégories) doivent être pris en compte pour constituer les séries.

Par example, le groupe H dans l’image ci-dessus est configuré pour sélectionner les deux premiers concurrents de chaque catégorie H*. Les concurrents qualifiés seront répartis dans deux séries, A et B, qui correspondent à des circuits de l’étape suivante.

Cliquez sur Actualiser pour générer et voir les séries. Cliquez sur Exporter pour exporter les séries générées dans un fichier.

Geco génère une liste de départ au format CSV qui peut être directement importé dans l’étape suivante.
