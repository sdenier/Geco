Pré-requis

Pour faire tourner Geco, vous avez besoin :

  • d’un JRE récent (Java Runtime Environment, version 6+)
  • d’une archive Geco qui contient l’application (fichier jar), différents fichiers modèles, la documentation, les bibliothèques natives de communication série
  • des pilotes SPORTident
    • Windows : voir le site SPORTident
    • Linux : les noyaux récents reconnaissent la puce utilisée par les boîtiers maîtres, plug’n’play !
    • Mac OS X : voir la FAQ pour une installation “à la main”.

Pour lancer Geco, exécuter le fichier geco.jar. Il devrait se lancer directement si votre JRE est bien configuré.
