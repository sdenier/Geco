Opérations en ligne de commande

Pour lancer Geco depuis la ligne de commande: java -jar gecoXXXX.jar

Options :

  • --startdir <DossierDeDonnees> : ouvre Geco directement sur l’étape sélectionnée, sans passer par le lanceur
  • --leisure : active le mode loisir pour LiveMap, affichant des données simplifiées sans temps de course pour des coureurs loisirs (non chronométrés)

Autres programmes accessibles depuis la ligne de commande :

  • java -jar gecoXXX.jar net.geco.GecoPools <DossierDeDonnees> : lance la fusion des poules en batch
  • java -jar gecoXXX.jar net.geco.live.GecoLive : ouvre le serveur GecoLive pour l’animation
